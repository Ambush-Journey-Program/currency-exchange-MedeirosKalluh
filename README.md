## Briefing

Your challange is to build this Dollar to Reais currency exchanger.

You can use a local declartion of the Exchange Rate for this challenge. By default the value to show on the amount is 1 Dollar.

## **Your users should be able to:**

- [ ] View the optimal layout for the app depending on their device's screen size
- [ ] Have the exchange values working bothsides (Dollar to Real and Real to Dollar)
- [ ] **Bonus**: Have other currencies listed on the exchange
- [ ] **Bonus**: Connect to an external API to get the updated exchange rates
